## Overview

This project is a fully functional custom keyboard built using TouchGFX for industrial applications.
It is designed as a modular and reusable UI component and tested on real STM32 hardware.


## Features

* Uppercase & Lowercase keyboard
* Numeric keypad
* Special characters support
* Password input handling
* Email input support
* Calibration data input
* Modular design using custom containers


## Technologies Used

* TouchGFX
* STM32
* C++


## Hardware

* Board: STM32H750B-DK
* IDE: STM32CubeIDE


## Use Case

Designed for real-time industrial systems requiring reliable and flexible user input.
